# تطبيق تعلم اللغات

تطبيق ويب تفاعلي لتعلم اللغات باستخدام البطاقات التعليمية.

## المميزات
- ✅ بطاقات تعليمية تفاعلية
- ✅ نظام التكرار المتباعد
- ✅ تتبع التقدم
- ⏳ تمارين متنوعة (قيد التطوير)
- ⏳ ألعاب تعليمية (قيد التطوير)

## التقنيات المستخدمة
- HTML
- CSS
- JavaScript

## كيفية التشغيل
1. انسخ المشروع: `git clone https://github.com/username/my-language-app.git`
2. افتح ملف `index.html` في المتصفح
